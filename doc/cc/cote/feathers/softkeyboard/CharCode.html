CharCodeConstants
 
Packagecc.cote.feathers.softkeyboard
Classpublic class CharCode
InheritanceCharCode Object

The CharCode class is an enumeration class containing constants associated to common characters or keys. Each constant contains the charCode value of the character in question. All ASCII characters are currently included as well as codes for the special SWITCH_LAYOUT, DONE and SPACER keys.

These constants are particularly useful when building Layout classes and key collections or when checking input received in a KeyEvent object.

See also

cc.cote.feathers.softkeyboard.layouts
cc.cote.feathers.softkeyboard.keycollections
